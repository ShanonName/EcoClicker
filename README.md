# Eco Clicker

`Un juego clicker con tematica de salvar el planeta`

### Requerimentos
- Python3
- Django

Para instalar los requisitos puedes ejecutar
`pip install django`

### Ejecutar la pagina

Estando en el directorio del repositorio ejecutar
`python manage.py runserver`
